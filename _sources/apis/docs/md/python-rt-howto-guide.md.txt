# How-To Guide for Alfa Python

- `Q1: Setup Alfa Python runtime on development environment`_

Please contact info@alfa-lang.io for any questions not covered below.

#### Q1: Setup Alfa Python runtime on development environment

The Alfa Python runtime is supplied as a pip installable package that can be installed
using following command.
```
pip install alfa-python-0.8.0.tar.gz
```
