# How-To Guide for Alfa TypeScript

- `Q1: Setup Alfa CSharp runtime on development environment`_

Please contact info@alfa-lang.io for any questions not covered below.

#### Q1: Setup Alfa CSharp runtime on development environment

The Alfa CSharp runtime is supplied as a NuGet package that can be installed using following command.
```
nuget add Alfa.Net.Runtime.0.8.0.nupkg  -Source $HOME/.nuget/packages/
```
