# Runtime APIs & Guides

Alfa provides runtime APIs for code it generates for target languages.

The following sections contain documentation for each of the target
languages generated from the language specific tools.

### Alfa Java Runtime
Javadoc documentation for the Alfa Runtime is available at the
link below. This includes the list of the primary classes and interfaces
in the Alfa Java runtime that users will interact with.

#### JavaDocs

>[Version 0.8 (Latest)](https://alfa-lang.io/apis/java/apidocs/index.html)
