split
=====

Split the input string by the given token.

__Parameters:__

|No.     |Type      |Comments    |
|--------|----------|------------|
| 1      |string    |Main string |
| 2      |string    |Split token |


__Returns:__

`list<string>` - List of strings split by the token.

__Usages:__

>- `split( val : string, token : string) : list<string>`
