put
===

Puts a new item (key, value) into a :ref:`mapVectorType`.

__Parameters:__

|No.|Type  |Comments|
|--------|----------|--------|
|1|:ref:`mapVectorType`|The map object we wish to put a new entry|
|2|Value of type `K`|Key to be used for the new entry|
|3|Value of type `V`|Value to be used for the new entry|

__Returns:__
`void` - no result.

__Usages:__

>- `put( v:map<K, V>, key:K, val:V ) : void`
