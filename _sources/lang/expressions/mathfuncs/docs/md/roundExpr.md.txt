round
=====

Round the number to the nearest value.

__Parameters:__

|No.|Type  |Comments|
|--------|----------|--------|
| 1 |:ref:`floatType` or :ref:`doubleType`|Valued to be round'ed|

__Returns:__

:ref:`intType` - If input is :ref:`floatType`

or

:ref:`longType` - If input is :ref:`doubleType`


__Usages:__

>- `round( v : double ) : long`
>- `round( v : float ) : int`
