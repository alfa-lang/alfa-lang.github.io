seconds
=======

Get the 'seconds' value from a :ref:`datetimeType`.

__Parameters:__

|No.|Type  |Comments|
|---|------|--------|
|1  |:ref:`datetimeType`|Value to extract seconds from|

No parameters accepted.

__Returns:__

`short(0,999)` - A :ref:`shortType` value representing the seconds.


__Usages:__

>- `seconds( d:datetime) : short`
