isNone
======

Given an optional value, checks if the value is empty or 'None'.

__Parameters:__

|No.|Type  |Comments|
|--------|----------|--------|
|1|Optional type|Any type marked as optional |

__Returns:__
`boolean` - `true` if the value is `None`.

__Usages:__

>- `isNone( v : T ? ) : boolean`
