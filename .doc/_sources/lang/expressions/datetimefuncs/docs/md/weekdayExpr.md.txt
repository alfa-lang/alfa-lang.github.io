weekday
=======

Get the weekday value as a number from a :ref:`dateType` or :ref:`datetimeType`.

__Parameters:__

|No.|Type  |Comments|
|---|------|--------|
|1  |:ref:`dateType` or :ref:`datetimeType`|Value to extract weekday from|

__Returns:__

`short(1,7)` - A :ref:`shortType` value representing the weekday.


__Usages:__

>- `weekday( d:date) : short`
>- `weekday( d:datetime) : short`
