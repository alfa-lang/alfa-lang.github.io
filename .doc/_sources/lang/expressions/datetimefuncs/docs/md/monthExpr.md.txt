month
=====

Get the 'month' value from a :ref:`dateType` or :ref:`datetimeType`.

__Parameters:__

|No.|Type  |Comments|
|---|------|--------|
|1  |:ref:`dateType`, :ref:`datetimeType` or :ref:`durationType`|Value to extract month from|

No parameters accepted.

__Returns:__

`short(1,12)` - A :ref:`shortType` value representing the month.


__Usages:__

>- `month( d:date) : short`
>- `month( d:datetime) : short`
