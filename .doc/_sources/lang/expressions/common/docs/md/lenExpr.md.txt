len
===

Returns the length of the object.

__Parameters:__

|No.|Type  |Comments|
|--------|----------|--------|
|1|Object of type :ref:`stringType`, :ref:`mapVectorType`, :ref:`listVectorType` or :ref:`setVectorType`|Object to check then length of|

__Returns:__
`int` - The length

__Usages:__

>- `len( v:list<T> ) : int`
>- `len( v:set<T> ) : int`
>- `len( v:map<K, V> ) : int`
>- `len( v:string ) : int`
