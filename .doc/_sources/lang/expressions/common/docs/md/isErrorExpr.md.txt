isError
=======

Given an `try` value, checks if the value is an error.

__Parameters:__

|No.|Type  |Comments|
|--------|----------|--------|
|1|Try type|Value of type type `try<T>`|

__Returns:__
`boolean` - `true` if the value is contains an error, `false` if its a valid value of type `T`.

__Usages:__

>- `isError( v : try< T >) : boolean`
