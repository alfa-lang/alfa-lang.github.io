# How-To Guide for ALFA CSharp

- `Q1: Setup ALFA CSharp runtime on development environment`_

Please contact info@schemarise.com or goto [www.schemarise.com](http://www.schemarise.com) for any questions not covered below.

#### Q1: Setup ALFA CSharp runtime on development environment

The ALFA CSharp runtime is supplied as a NuGet package that can be installed using following command.
```
nuget add Alfa.Net.Runtime.0.8.1-RC1.0.nupkg  -Source $HOME/.nuget/packages/
```
