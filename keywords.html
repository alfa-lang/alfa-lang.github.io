Keywords¶

This is a list of all ALFA keywords.

Any keyword can be used as an identifier by surrounding it with backticks. For example as record is a keyword, to use it as a field name it needs to be declared as `record`.

  • Annotations
  • entity
  • enum
  • enum< C1, C2, … >
  • fields
  • fragment
  • include
  • key
  • map< K, V >
  • namespace
  • record
  • seqVectorType
  • service
  • set< T >
  • trait
  • tuple< :T1, :T2, .. >
  • typedefs
  • union
  • union< :T1, :T2, ..>
